# JavaScript中的in

## HTML5 Canvas

### 1.1 概念
- HTML5的**canvas元素**使用**JavaScript**在网页中绘制图形
- **画布**：矩形，可以控制每一像素

### 1.2 创建Canvas元素
```html
<canvas id="myCanvas" width="200" height="100"></canvas>
```

### 1.3 通过JavaScript绘制
- canvas本身**没有绘图能力**，绘制工作在JavaScript中完成
